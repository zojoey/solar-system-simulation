# 纹理图片说明

此文件夹用于存放太阳系行星的纹理图片。您需要添加以下纹理图片文件：

1. sun.jpg - 太阳纹理
2. mercury.jpg - 水星纹理
3. venus.jpg - 金星纹理
4. earth.jpg - 地球纹理
5. mars.jpg - 火星纹理
6. jupiter.jpg - 木星纹理
7. saturn.jpg - 土星纹理
8. uranus.jpg - 天王星纹理
9. neptune.jpg - 海王星纹理

您可以从以下资源获取这些纹理图片：

- NASA官方网站: https://www.nasa.gov/multimedia/imagegallery/
- Solar System Scope: https://www.solarsystemscope.com/textures/
- Planetary Texture Maps: https://planetpixelemporium.com/planets.html

下载后，请将图片文件放置在此文件夹中，并确保文件名与代码中引用的名称一致。
